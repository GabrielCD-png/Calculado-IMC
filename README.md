# Calculadora de IMC (Índice de Massa Corporal)

Este projeto é uma aplicação web funcional desenvolvida para calcular o IMC de um utilizador com base no seu peso e altura. O objetivo principal foi exercitar a manipulação do DOM e a integração entre as tecnologias base da web (HTML, CSS e JavaScript).

## Funcionalidades
* **Entrada de Dados**: Campos de `<input>` configurados para receber valores numéricos de peso e altura.
* **Cálculo em Tempo Real**: Processamento matemático utilizando a fórmula $IMC = \frac{peso}{altura^2}$ através de JavaScript.
* **Classificação Automática**: Exibição da categoria de saúde (Abaixo do peso, Normal, Sobrepeso ou Obesidade) utilizando estruturas condicionais `if/else`.
* **Interface Responsiva**: Design adaptável para diferentes tamanhos de ecrã, garantindo uma boa experiência do utilizador (UX).

## Tecnologias Utilizadas
* **HTML5**: Estruturação semântica da página e dos formulários.
* **CSS3**: Estilização visual, incluindo o uso de seletores e propriedades de cor e alinhamento.
* **JavaScript**: Lógica de programação para captura de valores, cálculos e atualização dinâmica do conteúdo.

## Como Executar o Projeto
1. Clone este repositório para a sua máquina local.
2. Abra o ficheiro `index.html` em qualquer navegador moderno.
3. Insira os dados solicitados e clique no botão **Calcular**.
